How the Phrase Sheet Translator works

How does it work?

The tool automatically finds and translates all sheets with the correct structure. A sheet is valid if it contains these headers:

  • Cell A2: "Key"
  • Cell B2: "Original phrase"
  • Cell C2: "Translation"

In matching sheets, values from column B are translated using the database and written to column C. Search is case-insensitive.

File format

  • File to translate: Must contain at least one sheet with the header structure described above.
  • Translation database file: Column A should contain the source text (e.g. English), and column B its translation.

How the SpecsValue Translator works

How does it work?

This tool has complex multi-step logic depending on the value in column B:

  • When col. B is "features" or "full_features": Translates text found between tags > oraz <. Search is case-sensitive. Missing phrases are reported.
  • When col. B is "transmission": Only translates the word "Automatic" (case-insensitive).
  • In all other rows: Translates a list of special phrases (air suspension, coil suspension, seats, seat, air, coil), ensuring longer phrases are translated before shorter ones to avoid errors.

File format

  • File to translate: Must contain a sheet named SpecsValue.
  • Translation database file: Column A should contain the source text (e.g. English), and column B its translation.
